# RuneBingo API

REST API for RuneBingo, an application that lets Old School RuneScape players create and manage bingo events.

## Project setup

Install dependencies

```bash
npm install
```

Copy the `.env.example` file to `.env` and fill in the necessary values.

```bash
cp .env.example .env
```

## Compile and run the project

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Run tests

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
